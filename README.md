# KH2FM-Rando-Launcher
The KH2FM Rando Launcher is a simple way to open all of the necessary components for KH2 rando all at once. 

Upon a successful setup, double clicking your .bat file will launch the following all at once:

1. KHTracker
2. Livesplit
3. Seed Generator
4. build_from_mm
5. Epic Games Launcher
6. Mods Manager

# Instructions

1. Go to releases and download the .bat file.
2. Move this .bat file to a place you can find it. I personally keep it on my desktop since I play rando a lot.
3. Find the folder that has your KHTracker. Place the path of that folder where you're instructed to in the .bat file. It should replace both the instruction and the parentheses BUT NOT the cd.
4. Repeat step 3 with your KH2Randomizer.exe (seed generator).
5. Repeat step 3 with your OpenKh.Tools.ModsManager.exe (mods manager)
6. Repeat step 3 with build_from_mm.exe.
7. Repeat step 3 with Livesplit
8. Repeat step 3 with Epic Games.
